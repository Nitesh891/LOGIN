Process Management

(1.) Manages running applications (processes) and ensures efficient CPU utilization.

(2.) Handles multitasking, scheduling, and process execution.

Memory Management:

(1.) Allocates and deallocates RAM to programs as needed.

(2.) Uses virtual memory to extend RAM when required.

File System Management:

(1.) Organizes files and directories for easy access and security.

(2.) Provides read, write, and execute permissions for data.

Device Management:

(1.) Controls input/output devices like keyboard, mouse, printer, etc.

(2.) Uses drivers to enable communication between hardware and software.

Security & User Management:

(1.) Ensures system security with authentication (passwords, biometrics).

(2.) Manages multiple user accounts with different access levels.
